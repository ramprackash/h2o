Application developers are welcome to use the database hosted on this site using the following RESTful GET APIs
provided:


http://h2o.ng.bluemix.net/stations - To get the list of measuring stations for reservoirs in the database. This also provides the position in latitude/longitude format and other information about each station.
(The "id" returned by this API should be used in other APIs mentioned below.)

http://h2o.ng.bluemix.net/stationhistory - To get the historical water-level for 10 years for all stations.
http://h2o.ng.bluemix.net/stationhistory?id=<STNID> - To get historical water-level data for 10 years for the given station id.
http://h2o.ng.bluemix.net/stationhistory?id=<STNID>&year=<YYYY> - To get historical water-level data for the specified year for the given station id.



WARNING: The data served by this website may not be up-to-date.

It is a convienent aggregation of data, the most recent of which can be obtained by browsing various pages of the CDEC website - http://cdec.water.ca.gov
